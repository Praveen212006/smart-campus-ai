# 🎓 Nova Admission AI Portal

An AI-powered College Admission Prediction and Recommendation System that helps students predict admission chances and find suitable colleges based on their academic profile.

---

## 🚀 Project Overview

Nova Admission AI Portal uses Artificial Intelligence and Machine Learning to analyze student academic details and provide:

- Admission probability prediction
- College recommendations
- Cutoff analysis
- Course suggestions
- Admission guidance

The platform provides an interactive dashboard for students and parents.

---

## ✨ Features

### 👨‍🎓 Student Module

- Student registration and login
- Personal profile management
- AI admission prediction
- Recommended colleges
- Cutoff comparison
- Application status tracking


### 👨‍👩‍👧 Parent Module

- View student academic details
- Monitor admission progress
- Check recommended colleges
- Track application status


### 🏫 College Information

- College details
- Department information
- Available courses
- Faculty details
- Campus events
- Gallery


### 🤖 AI Features

- Machine Learning based prediction
- Data analysis
- Admission chance estimation
- Smart college recommendation
